# Jon's Surfline 
Displays the current surf forecasts from different counties spread all across California🏄🏼‍♂️

Get information about:

 1. Surf Locations📍
 2. Surf conditions✅
 3. Swell height🔥
 4. Wind condition🌬
 5. Tide🌊
